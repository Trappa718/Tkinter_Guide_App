## Tkinter Guide App

Учебное настольное приложение на Python для изучения библиотеки **Tkinter**.  
Проект сочетает в себе **теорию**, **видеоуроки**, **визуальную библиотеку виджетов**, **тренажёр с задачами** и **справочные материалы** в одном современном интерфейсе на базе `flet`.

Репозиторий на GitHub: [`Tkinter_Guide_App`](https://github.com/Trappa718/Tkinter_Guide_App)

---

### Основные возможности

- **Интерактивная библиотека по Tkinter**
  - Структурированное оглавление по главам:
    - **Глава 1. Основы Tkinter** (первая программа, окно приложения)
    - **Глава 2. Виджеты** (кнопки, `Label`, `Entry`, `Frame`, `Listbox`, `Scrollbar`, `Combobox`, `Scale`, `Spinbox`, `Progressbar`, меню, `Notebook` и др.)
    - **Глава 3. Text** (многострочное поле, операции, стилизация)
    - **Глава 4. Treeview** (таблицы, деревья, сортировка, выделение строк)
    - **Глава 5. Окна** (создание окон, `messagebox`, диалоговые окна)
    - **Глава 6. Стилизация** (шрифты, цвета, курсоры, стили, темы)
    - **Глава 7. Canvas** (добавление и управление элементами, теги, события)
  - Для каждой темы предусмотрены иллюстрации (`Library_Images`, `Lesson_Images`).

- **Раздел с видеоуроками**
  - 15 обучающих видео по Tkinter:
    - кнопки, метки, поля ввода;
    - менеджеры компоновки `pack`, `place`, `grid`;
    - стили `ttk`, вкладки (`Notebook`), `Toplevel`;
    - работа с фоном окна, HTML в Tkinter;
    - создание виджетов по нажатию кнопки, контекстное меню и др.
  - Видео открываются во внешнем плеере (поддержка Windows / macOS / Linux).

- **Визуальная библиотека виджетов**
  - Галерея изображений виджетов и интерфейсных элементов (`Library_Images`).
  - Быстрый визуальный обзор того, как выглядят основные компоненты Tkinter.

- **Тренажёр / задачи**
  - Задачи по Tkinter с готовыми примерами в `Ready_Tasks/`.
  - Возможность отрабатывать навыки после просмотра теории и видео.

- **Страница “О программе”**
  - Встроенная документация (`MANUAL.pdf`, `README.txt`, `LICENSE.txt` в папке `About_Page`).
  - Информация об авторе и лицензии.

- **Современный UI**
  - Интерфейс построен на **Flet** (Material Design‑подобный внешний вид).
  - Боковая панель навигации (библиотека, видео, функции, тренажёр, о программе).
  - Переключение **светлой / тёмной темы**.

### Технологии

- **Язык**: Python 3.10+ (рекомендуется)
- **GUI‑фреймворк**: [Flet](https://flet.dev/) (поверх Flutter, но с использованием Python)
- **Тема курса**: Tkinter (стандартный GUI‑модуль Python)
- Дополнительно:
  - Локальные ресурсы: изображения (`Lesson_Images`, `Library_Images`), видео (`Video_Tkinter`), задачи (`Ready_Tasks`).
  - Сборка в `.exe` с помощью `PyInstaller` (см. `main.spec`, папки `build/`, `dist/`).

### Структура проекта

- **`main.py`** — основной файл приложения Flet (логика навигации, темы, страниц).
- **`logo.ico`** — иконка приложения.
- **`About_Page/`**
  - `MANUAL.pdf` — руководство пользователя.
  - `LICENSE.txt` — лицензионная информация.
  - `README.txt` — краткое описание раздела.
  - `about_card.png`, `card.png` — изображения для страницы "О программе".

- **`Ready_Tasks/`** — подготовленные задания по Tkinter (`Task_1.txt` и др.).

### Установка и запуск из исходников

1. **Клонировать репозиторий**

```bash
git clone https://github.com/Trappa718/Tkinter_Guide_App.git
cd Tkinter_Guide_App
```

2. **Создать виртуальное окружение (по желанию, но рекомендуется)**

```bash
python -m venv venv
venv\Scripts\activate  # Windows
# или
source venv/bin/activate  # Linux / macOS
```

3. **Установить зависимости**

Минимально необходимо установить `flet`:

```bash
pip install flet
```

При необходимости:

```bash
pip install pyinstaller
```

4. **Подготовить ресурсы**

Убедитесь, что в корне проекта существуют папки:

- `Video_Tkinter` с файлами `lesson_1.mp4` … `lesson_15.mp4`;
- `Lesson_Images` с изображениями `lesson_1.jpg` … `lesson_15.jpg`;
- `Library_Images` с файлами `t1.png` … и т.д.;
- `Ready_Tasks` с задачами (`Task_1.txt` и др.).

Если их нет, создайте каталоги и поместите в них свои материалы.  
При первом запуске некоторые функции автоматически создадут пустые каталоги и выведут подсказки в консоль.

5. **Запуск приложения**

```bash
python main.py
```

Приложение откроется в отдельном окне Flet (настольный режим).

---

### Сборка `.exe` (Windows)

Для создания исполняемого файла можно использовать `PyInstaller`.  
В репозитории уже есть пример спецификации: `main.spec`.

Пример базовой команды:

```bash
pyinstaller --noconsole --onefile --icon=logo.ico main.py
```

или с использованием `main.spec`:

```bash
pyinstaller main.spec
```

Собранный файл будет находиться в папке `dist/`.
